    Usage:
        prgm [OPTIONS] SUBCOMMAND [ARG] ...

        SUBCOMMAND                    subcommand

    Subcommands:
        add                           Add item
        show                          Show item
        rm                            Remove item
        ls                            List items

    Options:
        -h, --help                    print help
